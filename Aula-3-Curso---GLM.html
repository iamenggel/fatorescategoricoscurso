Premissas para anlises parmetricas dos dados:

  • Distribuio normal;
  • Varincia constante
  • Independncia

Existem vrias situaes onde os dados no apresentam essas caractersticas

  • As respostas so binria(0,1), ou contvis
  • Apresentam distribuio no-normal 1.

Solues para esse problema

Transformao dos dados

  • Induz aproximao para normalidade;
  • Estabiliza a varincia;
  • no garantia que uma transformao iro resolver esses problemas: pode haver uma troca entre homocedasticidade e linearidade e necessidade de aplicao de outro modelo 2.

Modelos lineares generalizados (GLM)

  • Os pesquisadores biolgicos podem especificar diretamente a distribuio de erros e a relao entre a mdia e a varincia;
    • Erro binomial (binrio)
    • Erro Poisson (contagens por unidade) 3.
